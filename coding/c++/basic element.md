### arithmetic
- the general form to write arithmetic expression:
	`X + Y = ANS;`

| proscess            | symbol/operator | example                                                              |
| ------------------- | --------------- | -------------------------------------------------------------------- |
| addition            | `+`             | `40.5 + 2.2 = 42.7` or `28 + 5 = 33`                                 |
| subtraction         | `-`             | `50.2 - 20 = 30.2` or `60 - 30 = 30`                                 |
| multiplication      | `*`             | `10.5*3 = 31.5` or `7*3 = 21`                                        |
| division            | `/`             | `10.0/3 = 0.3333` or `7/3 = 2`                                       |
| modulus (remainder) | `%`             | `29%9 = 2`<br><ul><li>never use the modulus with floating value</ul> |
