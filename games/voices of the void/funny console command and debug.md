[_reddit guide_](https://www.reddit.com/r/Voicesofthevoid/comments/14xqnds/updated_all_valid_console_commands/)

- _ariral_ - Returns an error message
- _ariral.pls_ - Triggers the ariral slapper event
- _alien/summon_alien_ - Sometimes spawns a cutout of an alien behind the next door the player opens. For best results, spam the command into the console multiple times
- _maxwell_ - Spawns maxwell (also known as 'Dingus') somewhere randomly. check [[#cat(dingus)]] for all th cat
- _gooseworx.rufus_ - Spawns rufus, an entity that will seek and kill the player
- _gooseworx.thiccfus_ - Spawns thiccfus, a variant of rufus that is slower, but much more dangerous
- _rufus.fuckoff_ - despawns both rufus and thiccfus
- _llama.saatana_ - Spawns a mostly harmless Perkele Llama plushie adjacent to a server. (usually it's one of the main servers found at base, but exiting to main menu will change witch server it spawns near.)
- *sauce.get* - Spawn sauce gift
- _madness.combat_ - Triggers the madness combat minigame when run the command during _the red sky event_
