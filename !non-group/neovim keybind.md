##### insert mode
- i
	- exit insert mode: esc
##### command panel
- :
##### save
- w (in command panel)
- save and exit: wq (in command panel)
##### quit
- q (in command panel)
##### copy/paste
- requirement: `wl-clipboard`(wayland) or `xorg-xclipboard`(X11)
- keybind
	- copy: y (yanked)
	- paste: p

##### select entire file to something
- gg, VG
##### delete selected
- d
### find text
- /
- n (go to next sequence)
- N (go to previosus sequence)
