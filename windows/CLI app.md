### scoop
- [scoop](https://scoop.sh/)
	- package manager
	- how to install:
		- `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`
		- `Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression`
		- optional: `scoop add bucket extras`
- #### git (important) (main)
	- git / github for manage git stuff
	- `scoop install git`
- ### main bucket
	- imagemagick
		- image manipulation
		- `scoop install imagemagick`
	- ffmpeg
		- audio manipulation
		- `scoop install ffmpeg`
