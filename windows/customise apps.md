### apps
- flow launcher

- komorebi
	- windows tiling manager
	- available in:
		- scoop ([[CLI app|extras]]) `scoop install komorebi`
- whkd
	- light hotkey apps for komorebi
	- available in:
		- scoop ([[CLI app|extras]]) `scoop install whkd`
- yasb reborn
	- linux like bar
	- available in:
		- scoop ([[CLI app|extras]]) `scoop install yasb`
- pywal / winwal (python) [(guide)](https://github.com/scaryrawr/winwal)
	- color grep for desktop customization (used for yasb bar and terminal
- starship
	- customizable promt for any shell or terminal decoration
	- available in:
		- scoop (main) `scoop install starship`
