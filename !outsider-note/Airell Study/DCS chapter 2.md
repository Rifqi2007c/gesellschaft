==Data Type==:
	- Semua variable (pembolehubah) dan constant (pemalar) perlu diisytiharkan sebelum boleh digunakan dalam program
	- bagi mengisytihar, perlu kenalpasti kategori data dan namakan dengan secara unik
	- Contoh isytihar pembolehubah dan pemalar:
		- *data_type nama_pembolehubah*
		- int age;
		- double totalPrice
		- *const*(constant) double PI = 3.142

==Ungkapan Aritmetik:==
	Terdapat beberapa operator bagi ungkapan aritmetik (basically math ah):
		- Tambah +
		- Tolak -
		- Darab *
		- Bahagi /
		- Modulus %
	Operator ini akan mengikut konsep BODMAS

==Input Output Statement:==
	 Untuk input data pada pembolehubah, perlu menggunakan command:
		 cin >> masukkan pembolehubah
	Untuk memapar output, perlu menggunakan command:
		cout << "statement";
		or
		cout << output variable name;

==How to write a c++ program==
	- Start with common line and header file,
	- Continue with the main function.
	- Write declaration statements.
	- Write the executable statements that include the cin, cout and the process statements in between the open and close braces that show the limitation of the C++ program.
