### basic computer operation

- process
	manipulate data
- output
	produce result

### computer language
computer can only understand in machine language also known as _binary number or binary code_ which is _0 and 1_ (binary digit)(bits)
- machine language = binary code
- binary code = 0 and 1
- 8 bits = a byte
### programming languange type

- high-level language
	- c++
	- java
	- and more like it
### c++ relationship
- program

	- assembler + compiler + interpeter c}=> assembly + machine code => mahine code
- compiler
	translate instruction in high-level language all at one time
- interpeter
	translate instruction in high-level language one at a time

### comments
- not important in any program but they are usually usefull for documentation
	- can be written in any way
	- can be use to store programmer future reference
- symbol
	- `//` comment in a single line in between the `/`
	- `/* */` comment in a multiple line in a group
		- `/*` (open) open and put comment inside
		- `*/` (close) close comment and continue writing code
### pre-process directive
- header file in c++
- line of code that start with `#include<header-file>`
- the most common and important header `#include<iostream>`
	- other example:
		- `#include<math.h>`
### function
- a block of code that carries out specific task
- all function must start with `main()`
- generally main function start with fuction type `void` or `int` 
- its adviseable when write main function `int` and the line `return 0` must be put at the end of the function
