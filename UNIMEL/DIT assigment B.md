1. pengenalan kepada perisian aplikasi
Perisian aplikasi (Aplikasi) ialah sejenis perisian yang melaksanakan fungsi khusus untuk pengguna akhir dengan berinteraksi secara langsung dengannya. Satu-satunya tujuan perisian aplikasi adalah untuk membantu pengguna dalam melakukan tugas-tugas tertentu.

Visual Studio Code (VSCode) ialah perisian aplikasi penyuntingan kod sumber yang dibangunkan oleh Microsoft untuk Linux, macOS dan Windows. Kod Visual Studio menyediakan ciri seperti penonjolan sintaks, pelengkapan kod, petikan kod, pemfaktoran semula kod, penyahpepijatan dan Git. Microsoft mengeluarkan kod sumber Visual Studio Code dalam repositori GitHubnya di bawah lesen MIT (Code - OSS), manakala binari yang dibina oleh Microsoft tidak dikeluarkan di bawah lesen MIT dan merupakan perisian proprietari. Visual Studio Code mula diperkenalkan pada 29 April 2015, oleh Microsoft pada persidangan Build 2015. Versi pratonton dikeluarkan tidak lama selepas itu.

2. peranan dan tujuan dalam bidang IT
Perisian aplikasi adalah perisian yang mengandungi aturcara yang dibangunkan untuk memenuhi kehendak tertentu pengguna. Perisian aplikasi terbahagi kepada dua bahagian iaitu perisian asas dan perisian termaju. Perisian asas adalah perisian yang digunakan untuk meningkatkan produktiviti pengguna manakala perisian termaju pula adalah perisian untuk tujuan khusus dan memerlukan kepakaran untuk menggunakannya. Dengan bantuan komputer, pengguna boleh melaksanakan berbagai operasi yang dahulunya hanya boleh dilaksanakan oleh pakar yang terlatih.

Visual Code Studio memberikan kesenagan dan bantuan bagi kerja-kerja IT seperti Pembangun Perisian (Software Devloper) dengan alat-alat seperti pengedit kod (code editor), pengurus file, Terminal Bersepadu (Intergrated terminal) dan Penyahpepijat (Debugger).

3. Fungsi utama dan fungsi lanjutan
Fungsi perisian  ialah membantu pengguna dalam menyelesaikan tugasan tertentu, Ia terutamanya akan memberi manfaat kepada pengguna akhir dengan membantu mereka melakukan aktiviti khusus dalam pelbagai industri, termasuk pendidikan, perniagaan dan hiburan. Microsoft Word, sebagai contoh, ialah perisian aplikasi yang biasa digunakan yang membolehkan pengguna mencipta, mengedit, memadam dan melakukan tindakan lain pada dokumen Word. Selain itu, perisian aplikasi boleh membantu mengurus dan memanipulasi data. Perisian aplikasi digunakan dalam perniagaan untuk mengurus dan mengubah pangkalan data pekerja, pelanggan dan lain-lain.

Visual Studio Code (VSCode) berfungsi sebagai Integrated Development Environment (IDE) yang mempunyai kemudahan seperti Editor Kod (Code Editor) yang boleh digunakan dalam pelbagai Bahasa Pengaturcaraan (Programming Language) seperti c++, java, python dan lain-lain. Kemudahan lain yang disediakan oleh Visual Studio Code (VSCode) ialah Terminal Bersepadu (Intergrated terminal) dengan fungsi untuk membuka Terminal tanpa keluar dari aplikasi tersebut dan persekitaran editor kod.

4. Keperluan sistem komputer
Setiap perisian aplikasi memerlukan keperluan asas atau dikenali sebagai prasyarat dari perkakasan seperti kuasa pemprosesan CPU, jumlah saiz RAM dan jumlah saiz Storan Sekunder seperti Hard Drive. Prasyarat ini dikenali sebagai keperluan sistem (komputer) dan sering digunakan sebagai garis panduan yang bertentangan dengan peraturan mutlak. Kebanyakan perisian mentakrifkan dua set keperluan sistem iaitu minimum dan disyorkan. Keperluan ini diperlukan supaya perisian aplikasi dapat dijalan kan dengan cekap dan lancar.

Visual Studio Code mensyorkan pengguna mempunyai 1.6Ghz kuasa pemprosesan CPU, 1GB jumlah saiz RAM dan 500 MB jumlah saiz Storan Sekunder. syarat-syarat ini dikira ringan bagi banayak sistem komputer kini.

5. Kaedah mendapatkat perisian
Perisian boleh diperoleh dalam beberapa cara yang berbeza. Antara cara mendapatkan perisian ialah dari Perisian Komersial Luar Rak (COTS) yang melibatkan pembelian penyelesaian perisian pra-bina daripada vendor pihak ketiga. Contohnya termasuk Microsoft Office dan Adobe Photoshop. Selain itu, perisian boleh di perolehi dari Perisian Percuma (Freeware) yang melibatkan pengguna memuat turun dari laman web dan memasang perisian secara percuma tetapi pengguna perlu mematuhi syarat dan had pelesenan dari pembangun atau syarikat. Seteryusnya, perisian boleh diperolehi perisian dari Perisian Sumber Terbuka (Open Source Software) yang melibatkan syarikat memanfaatkan perisian sumber terbuka, yang tersedia secara percuma dan boleh diubah suai untuk memenuhi keperluan pengguna. Contohnya termasuk sistem pengendalian Linux, pelayan web Apache dan pangkalan data MySQL.

Visual Studio Code ialah Perisian Percuma dan boleh di muat turun dari laman web aplikasi tersebut.
Laman web Visual Studio menyediakan cara persidiaan untuk menggunakaannya untuk setiap Bahasa Pengaturcaraan (Programming Language) dan pelbagai funsi lain.

6. Empat cara Visual Studio Code meningkatkan produktiviti kerja IT
Visual Studio Code mempunyai banyak fungsi khas yang boleh meningkatkan produktiviti kerja IT, berikut adalah beberapa fungsi tersebut. ==Pertama==, Visual Studio Code mempunyai Editor Kod (Code Editor) yang boleh digunakan dengan pelbagai bahasa pengaturcaraan, termasuk C, C#, C++, Fortran, Go, Java dan lain. Ia membolehkan pekerja Pembagun perisian (Software Devloper) membina sesuatu perisian dengan apa-apa Bahasa Pengaturcaraan (Programming Language) yang diperlukan atau diplih oleh seorang Pembangun Perisian. ==Kedua==, Visual Studio Code menyediakan fungsi penyahpepijat Terbina Dalam (Built-in) yang direka untuk meningkatkan proses pembangunan. Ia menyediakan sokongan asli untuk menyahpepijat aplikasi Node.js, manakala penyahpepijat tambahan untuk bahasa pengaturcaraan lain boleh dipasang melalui sambungan. Penyahpepijat membenarkan pembangun melampirkan pada proses yang sedang berjalan dan melangkah melalui kod sumber baris demi baris semasa pelaksanaan, menawarkan pandangan terperinci aliran program. 

==Ketiga==, Visual Studio Code membenarkan pengguna membuka satu atau lebih direktori, yang kemudiannya boleh disimpan dalam ruang kerja untuk digunakan semula pada masa hadapan. Ini membolehkan ia beroperasi sebagai Editor Kod agnostik bahasa untuk mana-mana bahasa. ==Keempat==, Visual Studio Code menyediakan terminal intergrasi (intergrated) berciri penuh yang dibuka pada akar ruang kerja semasa, membolehkan pengguna menjalankan arahan shell tanpa meninggalkan persekitaran editor.

Kesimpulan
Kesimpulannya, perisian sistem dan perisian aplikasi dapat membantu kerja-kerja IT lebih mudah. Setiap pekerja IT haruslah menggunakan perisian dengan betul dan baik supayah tahap produktiviti meningkat dan ia juga boleh memberi impak kepada pengguna biasa. Disamping itu, kerja-kerja IT yang penting untuk dunia kita haruslah dilakukan dengan baik supaya masalah IT seperti masalah sekuriti, masalah sistem dan masalah yang boleh menimbulkan kemarahan pengguna biasa dapat dielakkan.
